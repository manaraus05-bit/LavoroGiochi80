# LavoroGiochi80
 
## Sviluppo grafica

### Definire stile grafico

- Pixel art
- Stile cartoon con colori saturi freddi
- Stile laterale 2D
- Animazioni chiare e brevi

### Sprite di player e nemici alieni

Dimensione sprite: **48x48 px**

Animazioni previste:

- Fermo: 2 frame
- Camminata: 4 frame
- Attacco base: 3 frame
- Attacco speciale: 4 frame
- Colpito: 1 frame
- Morto: 4 frame

## HUD

Elementi dell’interfaccia:

- Barra vita
- Punteggio
- Numero vite
- Barra energia speciale

## Livelli

1. **Strada buia illuminata da lampioni**  
   Con in secondo piano uno scorcio di città.

2. **Metropolitana abbandonata**  
   Illuminazione minima e colori vividi.

3. **Laboratorio alieno**  
   Luogo in cui vengono prodotti gli alieni, con ambientazione tecnologica e futuristica, in contrasto rispetto ai livelli precedenti.
